## ✨ Features at a Glance  

- 💾 **Save with Context** — Stores snippet + file path & line number.  
- 📚 **Snippet Explorer** — Sidebar tree view for browsing all snippets.  
- ⚡ **One-Click Insert** — Insert snippets right where your cursor is.  
- 🔗 **Go to Source** — Jump back to the original file instantly.  
- ✂️ **Copy & Delete** — Manage snippets with quick inline actions.  
- 🔍 **Search** — Filter snippets instantly in the sidebar.
